FlowP

FlowP is a data processing framework that supports both Dataflow and Functional Reactive programming paradigms in a variety of languages.

Designed with developers in mind, FlowP helps engineers create large scale data processing pipelines in a clean and simple way that makes sense and scales well.

Byte Reactor

Byte Reactor is a SaaS platform that exposes the power of FlowP as a remotely callable API.

It allows developers to very quickly create complex data processing flows that can be started with a simple HTTP POST and scale seamlessly with demand.

In addition to the data processing capabilities, Byte Reactor also exposes sophisticated metrics on the processing pipelines, allowing users to gain insight about their system, reduce cost by optimising usage and detect processing bottlenecks.
